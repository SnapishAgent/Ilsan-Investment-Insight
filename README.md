## 일산 신도시의 미래 투자 가치 탐색: 1차 신도시 재건축 지역과의 비교 분석
- 1989년 1기 신도시 건설 이후 형성된 부동산 가치 차이를 분석하여, 상대적으로 저평가된 일산 신도시의 투자 가치를 높일 방안을 탐구.
- 데이터 분석을 통해 주요 요인(아파트 실거래가, 일자리, 교통 접근성, 생활 인프라 등)이 투자 가치에 미치는 영향을 비교 및 예측.
